# B站视频字幕提取工具

## 概述

这是一个智能化的B站（Bilibili）视频字幕提取工具。除了基本的字幕提取功能外，**最大特色是支持关键词搜索和智能排序**——你可以输入"拍立得测评"等关键词，工具会自动搜索相关视频并按播放量、收藏数、点赞数等数据排序，让你轻松选择最热门或最适合的视频进行字幕提取。

## 使用场景

当你需要：
- **搜索B站视频**并按热度（播放量、收藏、点赞）筛选
- 从B站视频中提取字幕文本内容
- 将字幕导出为JSON、TXT、SRT或Markdown格式
- 批量处理多个B站视频的字幕提取
- 对热门视频字幕进行文本分析或翻译
- 创建视频字幕的备份存档

## 功能特性

- 🔍 **智能搜索**：支持关键词搜索B站视频
- 📊 **数据排序**：按播放量、收藏数、点赞数、发布时间等多维度排序
- ✅ **多格式导出**：支持JSON、TXT、SRT、Markdown格式
- ✅ **多语言支持**：自动提取视频的所有可用字幕轨道
- ✅ **批量处理**：支持批量提取多个视频的字幕
- ✅ **时间戳保留**：完整保留字幕的时间信息
- ✅ **错误处理**：完善的异常处理和错误提示
- ✅ **编码兼容**：自动处理Windows终端编码问题
- ✅ **交互式选择**：友好的命令行交互界面

## 工作流程

### 方式1：智能搜索+提取（推荐）⭐

**这是最智能的使用方式**，适合没有特定BV号的场景：

1. **输入关键词搜索**（如"拍立得测评"）
2. **选择排序方式**：
   - 按播放量排序（最多播放）
   - 按收藏数排序（最多收藏）
   - 按点赞数排序（最多点赞）
   - 按发布时间排序（最新发布）
   - 按弹幕数排序
3. **浏览搜索结果**：显示视频标题、UP主、播放量、收藏数等数据
4. **选择视频**：交互式选择一个或多个视频
5. **自动提取字幕**：批量提取选中视频的字幕

```bash
# 方式A：交互式搜索和提取（推荐）
python scripts/search_and_extract.py 拍立得测评

# 方式B：仅搜索，不提取
python scripts/search_videos.py 拍立得测评 click 15
```

### 方式2：直接提取（已知BV号）

如果你已经有BV号，可以直接提取：

**单个视频：**
```bash
python scripts/extract_single.py
```

**批量提取：**
```bash
# 编辑 scripts/video_list.txt 添加BV号
python scripts/extract_batch.py
```

### 文件说明

- `search_and_extract.py` - **智能搜索+提取工具**（推荐，一站式）
- `search_videos.py` - 视频搜索工具（仅搜索）
- `extract_single.py` - 单视频提取脚本
- `extract_batch.py` - 批量提取脚本
- `video_list.txt` - 批量提取的视频列表

## 智能搜索详细说明

### 排序方式

| 排序方式 | 说明 | 适用场景 |
|---------|------|---------|
| `click` | 按播放量排序 | 找最热门的视频 |
| `stow` | 按收藏数排序 | 找最被认可的内容 |
| `pubdate` | 按发布时间排序 | 找最新内容 |
| `dm` | 按弹幕数排序 | 找互动最多的视频 |
| `scores` | 按评分排序 | 找高质量内容 |

### 搜索结果展示

搜索后会显示每个视频的详细信息：
- 视频标题和BV号
- UP主名称
- 播放量、弹幕数、收藏数
- 发布时间
- 视频简介（前100字）

### 交互式选择

支持多种选择方式：
- **单个数字**：`1`（选择第1个视频）
- **多个数字**：`1,3,5`（选择第1、3、5个视频）
- **范围选择**：`1-5`（选择第1到第5个视频）
- **全部选择**：`all`（提取所有搜索结果的字幕）

## 输出格式说明

### JSON格式
- 完整保留字幕结构
- 包含视频元信息（标题、BV号、CID等）
- 每条字幕包含：开始时间、结束时间、文本内容
- 支持多字幕轨道

### TXT格式
- 易读的纯文本格式
- 时间戳显示为 HH:MM:SS.mmm
- 按字幕轨道分组
- 适合快速查看和文本处理

### SRT格式
- 标准字幕文件格式
- 可用于视频播放器
- 支持时间轴显示
- 通用性强，兼容性好

### Markdown格式
- 便于在Markdown编辑器中查看
- 支持代码高亮
- 结构清晰，易于阅读
- 适合文档化

## 技术实现

### API调用

使用B站公开API：
1. 视频信息API：获取视频标题、CID等基本信息
2. 字幕列表API：获取可用字幕轨道
3. 字幕内容API：下载具体字幕内容

### 错误处理

- 网络请求超时处理（10秒超时）
- API错误代码检查
- 文件写入异常处理
- 编码问题自动修复

### 依赖项

- Python 3.6+
- requests库（HTTP请求）
- json库（JSON处理）

## 注意事项

1. **网络连接**：需要稳定的网络连接访问B站API
2. **视频权限**：部分视频可能需要登录才能访问字幕
3. **字幕可用性**：不是所有视频都有字幕
4. **请求频率**：批量提取时建议添加适当延迟，避免频繁请求
5. **编码问题**：脚本已处理Windows终端编码问题

## 最佳实践

1. **先测试单个视频**：在批量提取前，先测试单个视频确保正常工作
2. **检查输出文件**：提取后检查生成的文件是否完整
3. **备份原始数据**：保留JSON格式的原始数据作为备份
4. **合理使用**：仅用于个人学习和研究，遵守B站使用条款

## 故障排除

### 问题：无法获取视频信息
- 检查网络连接
- 确认BV号是否正确
- 检查视频是否存在

### 问题：视频没有字幕
- 确认视频是否有字幕
- 部分UP主可能未上传字幕

### 问题：乱码问题
- 脚本已自动处理UTF-8编码
- 确保终端支持UTF-8显示

### 问题：需要登录的视频
- 使用 `bilibili_cookie_helper.py` 工具
- 添加登录Cookie到请求头
